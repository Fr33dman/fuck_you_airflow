v1.7.0 (16.06.2021)

Новый функционал:

- next_activity_prediction: функционал по предсказанию следующих шагов неоконченных цепочек (активностей и их временных меток) + отдельный tutorial
- отрисовка графов (MatplotlibPainter) и расчёт кординат при конвертации графа в bpmn-файл при помощи библиотеки grandalf (альтернатива graphviz-у, но работает несколько хуже по скорости и по качеству)
- Correlation miner (майнер, восстанавливающий граф по логу, в котором нет ID-колонки)
- High Order Proximity Embeddings, WeightedAdjacencyMatrix: эмбеддинги для цепочек, использующие графовое представление процесса
- добавлены отдельные метрики по графам: центральность (sberpm.graph_stats), adjacency_matrix, laplacian_matrix (методы класса Graph)

Также добавлено:

- В DataHolder теперь можно передать id_column=None (нужно для Correlation miner)
- В конвертацю графа в bpmn добавлен метод apply_dfg. Он принимает dfg-граф и записывает его в bpmn-файл. (Но это не будет строго bpmn-нотация, если у графа есть развилки.)
- работа с текстом: добавлена поддержка gensim версии 4
- rl: чуть более подробная документация
- в новый код и ряд старых классов добавлено описание типов агрументов и возвращаемых значений
- для ряда алгоритмов добавлены тесты
- Jupyter ноутбуки с примерами использования библиотеки перенесены в папку tutorials
- добавлен tutorial для ChartPainter и данные для него (InternationalDeclarations.csv)
- добавлен данный файл

Исправлено:
- исправлена ошибка в HeuMiner (он мог рисовать несуществующие рёбра)
- небольшой рефакторинг автоинсайтов и симуляции
- другие мелкие исправления
